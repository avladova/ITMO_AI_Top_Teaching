## Тема: «Ресэмплинг-методы: Бутстреп, метод складного ножа и построение доверительных интервалов»

### 1. Методические рекомендации

**Почему нам это нужно?**
В классической статистике для построения доверительных интервалов (ДИ) нам нужно знать форму распределения оценки (например, знать, что выборочное среднее распределено нормально). На практике распределения часто неизвестны, имеют тяжелые хвосты или мы используем сложные статистики (например, медиану, коэффициент асимметрии), для которых аналитических формул просто не существует. Ресэмплинг (передискретизация) позволяет обойти эту проблему, используя только ту информацию, которая уже есть в нашей выборке.

**1.1. Непараметрический бутстреп (Bootstrap)**
Идея Брэдли Эфрона (1979): наша исходная выборка $X = (x_1, ..., x_n)$ — это лучшая модель того, как выглядит генеральная совокупность. 
*   **Алгоритм:** 
    1. Из исходной выборки объема $n$ мы извлекаем $B$ новых выборок того же объема $n$ **с возвращением** (bootstrap-выборки).
    2. Для каждой $b$-й бутстреп-выборки вычисляем интересующую нас статистику $\hat{\theta}^*_b$ (например, среднее, медиану, дисперсию).
    3. В результате получаем бутстреп-распределение статистики: $\{\hat{\theta}^*_1, ..., \hat{\theta}^*_B\}$.
*   **Оценка стандартной ошибки:** Стандартное отклонение этого бутстреп-распределения является оценкой стандартной ошибки (SE) нашей статистики: $SE_{boot} = \sqrt{\frac{1}{B-1}\sum_{b=1}^{B}(\hat{\theta}^*_b - \bar{\theta}^*)^2}$.

**1.2. Бутстреп-доверительные интервалы (Percentile Method)**
Самый простой способ построения ДИ на основе полученного бутстреп-распределения. Мы просто берем квантили этого эмпирического распределения.
*   Для 95% ДИ: нижняя граница = 2.5-й процентиль, верхняя граница = 97.5-й процентиль массива $\{\hat{\theta}^*_b\}$.

**1.3. Метод складного ножа (Jackknife)**
Идея Джона Тьюки (1958). Более старый, но математически изящный метод, предшественник бутстрепа.
*   **Алгоритм:**
    1. Из исходной выборки поочередно удаляется ровно один элемент (формируется $n$ jackknife-выборок объемом $n-1$).
    2. Для каждой подвыборки без $i$-го элемента вычисляется статистика $\hat{\theta}_{-i}$.
    3. Вычисляется jackknife-оценка статистики: $\hat{\theta}_{jack} = \frac{1}{n}\sum_{i=1}^{n}\hat{\theta}_{-i}$.
*   **Оценка стандартной ошибки (Jackknife SE):** 
    $SE_{jack} = \sqrt{\frac{n-1}{n}\sum_{i=1}^{n}(\hat{\theta}_{-i} - \hat{\theta}_{jack})^2}$.

**Инструментарий Python:**
Для реализации потребуется `numpy` (особенно функция `np.random.choice` с параметром `replace=True` для бутстрепа), `pandas` и `matplotlib`/`seaborn` для визуализации.
